SystemInspector 使用 SOP

SystemInspector 是一款 Windows 系统检测与优化工具，用于快速查看硬件信息、驱动详情、系统状态、SSD 健康、屏幕 EDID、内置外部工具以及部分系统设置。

Windows 10 / 11 管理员权限运行 内置 6 款工具 支持命令行导出

1. 启动程序

  1. 双击运行 SystemInspector.exe。
  2. 如果系统弹出 UAC 权限确认，点击“是”。
  3. 程序打开后，通过左侧菜单切换不同信息页面。

程序启动后会先显示系统信息，再静默预加载 SSD 信息、屏幕详情、系统异常、电池日志和电源日志；加载期间界面仍可正常操作。

2. 功能模块

系统信息

查看电脑基础硬件与身份信息，包括主板、CPU、GPU、内存、SSD、BIOS、EC、TPM、序列号、UUID、网卡（含型号与MAC地址）和蓝牙地址等。BIOS 显示为"厂商 版本, 日期"格式。

系统状态

查看 Windows 版本、Build、系统架构、安装日期、最近启动时间、运行时长、激活状态、系统语言、时区、Secure Boot 和 BitLocker 状态。另外包含计算机名、域/工作组、登录用户、总/可用物理内存、虚拟内存、页面文件、引导设备、Hyper-V 状态、HAL 版本和电源计划等 msinfo32 风格摘要信息。

系统异常

查看设备管理器异常驱动/设备，以及最近 30 天 System 日志中的重大异常事件。重点筛选 Microsoft-Windows-WER-SystemErrorReporting 1001、System Error 1003 和 41、以及常见 Microsoft-Windows-Kernel-Power 41，并展示事件时间、来源、记录号和详细消息。

SSD 信息

查看每块 SSD 的 SMART 状态，包括型号、固件、序列号、接口类型、传输模式、温度、通电时长、通电次数、寿命估算、主机读写总量和设备标识。

启动项

查看电脑开机自动启动项目，包括注册表 Run、Run32 项和启动文件夹项目，并同步任务管理器启用/禁用状态。右键启动项可启用、禁用、删除或刷新列表；无实际命令或路径的 N/A 状态记录默认隐藏，也可以添加当前用户自定义开机启动项。

驱动详情

以设备管理器风格三级树形结构展示所有已签名驱动，按设备类别分组。每个设备下显示驱动版本、驱动日期和提供商，全部节点默认展开。数据来源于 Win32_PnPSignedDriver。

屏幕详情

读取当前活动显示器的 EDID 信息，展示显示器名称、显示器 ID、EDID 日期、版本、序列号、刷新率、校验和以及 EDID 原始十六进制数据。

电池日志

通过 powercfg /batteryreport 导出电池使用与容量 HTML 报告。页面会显示生成状态、保存路径、文件大小和生成时间；点击“详情”可在默认浏览器中打开报告，点击“刷新”会重新导出并覆盖程序目录下的 SystemInspector-BatteryReport.html。

电源日志

通过 powercfg /sleepstudy 导出睡眠与电源行为 HTML 报告。页面会显示生成状态、保存路径、文件大小和生成时间；点击“详情”可在默认浏览器中打开报告，点击“刷新”会重新导出并覆盖程序目录下的 SystemInspector-SleepStudy.html。

实用工具

内置 6 款硬件检测工具（CPU-Z、BatteryInfoView、Core Temp、HWiNFO64、CrystalDiskMark、USBTreeView），以图标卡片形式展示，点击可释放到程序目录并运行（已存在则直接启动）。另外提供 AMD / Nvidia 驱动官方下载链接，点击以默认浏览器打开。

系统设置

查看或调整 UAC、Windows 防火墙、安全中心提示、崩溃自动重启、完整内存转储、屏幕保护、电源策略和 Windows 自动更新等设置。

3. 命令行导出

程序支持通过 -Report 参数导出文本报告，适合批量检测或留档。

SystemInspector.exe -Report SYSTEM "D:\Temp\system-report.txt"
SystemInspector.exe -Report STATUS "D:\Temp\status-report.txt"
SystemInspector.exe -Report ACPI "D:\Temp\acpi-report.txt"
SystemInspector.exe -Report SSD "D:\Temp\ssd-report.txt"
SystemInspector.exe -Report EDID "D:\Temp\edid-report.txt"
类型 导出内容
SYSTEM 系统基础硬件信息。
STATUS Windows 版本、激活状态和安全状态。
ACPI CPU、GPU、逐条内存、BIOS/SMBIOS、主板、机箱和系统产品信息。
SSD SSD SMART 与健康状态。
EDID 当前活动显示器 EDID 信息。

如果在批处理脚本中连续导出，建议使用 start "" /wait SystemInspector.exe -Report ... 等待每次导出完成。

4. 使用注意

  • 程序需要管理员权限，部分信息在普通权限下可能无法读取。
  • 启动后的静默预加载会在后台生成电池日志和电源日志，并覆盖程序目录下的同名 HTML 报告文件。
  • 系统异常页面读取设备管理器异常状态和最近 30 天的 System 重点异常日志，避免完整扫描大型事件日志。
  • 屏幕详情只读取当前活动显示器，不读取历史连接过的显示器记录。
  • 电池日志和电源日志保存在程序 EXE 所在目录，刷新会覆盖同名 HTML 报告文件。
  • 导出报告可能包含序列号、UUID、MAC 地址等设备标识信息，请注意保存和传输安全。
  • 系统设置页面涉及 Windows 配置修改，操作前请确认当前需求。
